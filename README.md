# 🎬 Movie Recommender System

A content-based movie recommender system that suggests movies similar to the one you like using metadata such as genres, cast, and overview. Built with Python and Scikit-learn, it's a simple and effective way to explore new films.

---

## 📌 Features

- 🔍 Content-based filtering using metadata (genres, keywords, cast, crew)
- 📊 Cosine similarity for measuring movie similarity
- 🔎 Search and recommendation by movie title
- 🧼 Cleaned and preprocessed dataset
- 💡 Lightweight and easy to understand

---

## 🛠️ Tech Stack

- **Language:** Python
- **Libraries:** Pandas, NumPy, Scikit-learn, NLTK
- **Optional Add-ons:** Streamlit/Flask for UI, TMDB API for posters
